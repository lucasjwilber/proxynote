#mapChat
Share what you've seen! Add and reply to posts on a Google map, pinned to the location where they were created.

##Built With
* Firebase Auth
* Firestore Database
* Firebase Storage
* Glide (https://github.com/bumptech/glide)
* Transcoder (https://github.com/natario1/Transcoder)
